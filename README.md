# Блог на ASP.NET Core

Веб-приложение для ведения блога с возможностью регистрации, создания постов, комментариев и просмотра профилей пользователей.

## Что умеет

- Регистрация новых пользователей
- Добавление постов (название, краткое содержание, полный текст)
- Просмотр всех постов на главной странице блога
- Просмотр отдельного поста с комментариями
- Добавление комментариев к постам
- Страница пользователя со списком его постов
- Список всех зарегистрированных пользователей
- Удаление постов
- Удаление учётных записей
- Проверка существования пользователя при добавлении поста/комментария

## Как запустить

1. Установи .NET 9 (если ещё нет)
2. Склонируй репозиторий или скачай zip
3. Открой терминал в папке с проектом
4. Введи `dotnet run`
5. Открой браузер и перейди на `http://localhost:...` (Порт будет в консоли)

## Технологии

- C# и ASP.NET Core (Minimal API)
- Entity Framework Core + SQLite
- HTML, JavaScript (fetch, DOM)

## Структура проекта

- `Program.cs` - серверная часть, эндпоинты
- `script.js` - клиентская логика
- `*.html` - страницы (меню, регистрация, добавление поста, список пользователей, профиль, пост, список постов)

## Пара моментов

- База данных создаётся автоматически при первом запуске (`blog.db`)
- Пароли хранятся в открытом виде, так как проект учебный
- Для работы с постами и комментариями нужен существующий `userId`
- После добавления комментария или удаления пользователя, страница перезагружается
